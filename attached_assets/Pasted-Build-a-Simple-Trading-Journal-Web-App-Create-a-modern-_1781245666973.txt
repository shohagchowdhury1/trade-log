Automatic Calculations

Net P&L:

For Long:
((Exit - Entry) × Shares)

For Short:
((Entry - Exit) × Shares)

Result:

If P&L > 0 = Win

If P&L < 0 = Loss

R:R Calculation:

Reward = abs(Exit - Entry)

Risk = abs(Entry - Stop Loss)

R:R = Reward / Risk

Round to 2 decimals.
